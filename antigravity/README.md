# Antigravity — Multi-Model Agentic IDE

Antigravity is Google's agent-first IDE — a modified VS Code fork powered by Gemini with multi-model support for Claude, ChatGPT, and more. It's the environment where Limitless Stack agents run.

## Role in the Stack

Antigravity is the orchestration and execution layer. It provides:

- **Multi-model dispatch** — Switch between Gemini, Claude Opus/Sonnet, and GPT models based on task needs
- **Parallel agents** — Mission Control dispatches up to five agents working simultaneously on different parts of a project
- **Workspace integration** — The vault, repos, and tooling are all accessible from within Antigravity's workspace
- **Skill system** — Agents running in Antigravity can install skills (including this one) to learn protocols

For a 100-vertical platform, Antigravity is how you parallelize development work across apps — multiple agents working on different verticals simultaneously, each following the Limitless Stack protocol.

## Role in Self-Healing

Antigravity enables key self-healing capabilities:

- **Parallel fix candidates** — For high-severity bugs, Antigravity can dispatch multiple agent runs with varied approaches via Mission Control, producing multiple PRs for operators to compare. Cost is bounded (3x a single attempt) and gives operators better fix options.
- **Cross-model verification** — A fix generated by Claude can be reviewed by a Gemini agent for a second opinion, reducing false positives.
- **Bulk rollout** — When shipping the self-healing standard to new apps (Phase 3 of Dale's rollout plan), Antigravity can run parallel agents that each scaffold the canonical files into a different repo.

## Setup

Antigravity is installed as a desktop app (modified VS Code). The vault lives inside Antigravity's workspace. Any agent Antigravity dispatches should have the Limitless Stack skill installed.

## Models available

| Model | Best for |
|---|---|
| Gemini 3.1 Pro | General tasks, fast iteration |
| Claude Opus 4.6 | Complex multi-file reasoning, architecture decisions |
| Claude Sonnet 4.6 | Balanced speed and capability, diagnostic passes |
| GPT-OSS 120B | Alternative perspective, cross-validation |

## Integration points

- **Claude / Gemini / ChatGPT** — Antigravity dispatches any of these as the reasoning engine
- **CLAUDE.md** — Each agent reads the relevant CLAUDE.md for context
- **Obsidian** — The vault lives in Antigravity's workspace
- **Pinecone** — Agents search the corpus via Pinecone scripts in the workspace
- **NotebookLM** — Agents query topic notebooks for deep research
- **Paperclip** — Agents report task completion to Paperclip for tracking
- **Self-healing** — Parallel fix candidates, cross-model verification, bulk rollout of self-heal standard
